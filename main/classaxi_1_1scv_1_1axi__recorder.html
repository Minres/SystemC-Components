Detailed Description

template<typename TYPES = axi::axi_protocol_types>
class axi::scv::axi_recorder< TYPES >

The TLM2 transaction recorder.

This module records all TLM transaction to a SCV transaction stream for further viewing and analysis. The handle of the created transaction is storee in an tlm_extension so that another instance of the axi_recorder e.g. further down the path can link to it.
