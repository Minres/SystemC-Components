Detailed Description

template<typename TYPES = chi::chi_protocol_types>
class chi::lwtr::chi_lwtr< TYPES >

The TLM2 transaction recorder.

This module records all TLM transaction to a LWTR transaction fiber for further viewing and analysis. The handle of the created transaction is stored in an tlm_extension so that another instance of the chi_lwtr e.g. further down the path can link to it.
