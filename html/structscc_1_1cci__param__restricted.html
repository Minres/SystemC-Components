Detailed Description

template<typename T, cci::cci_param_mutable_type TM = cci::CCI_MUTABLE_PARAM>
struct scc::cci_param_restricted< T, TM >

extension of cci_param<T, TM> which automatically registeres a callback to restrict the valid values given to the parameter.

The parameter can be use with a restriction created by:

  • min_max_restriction(T min, T max)
  • min_max_excl_restriction(T min, T max)
  • min_restriction(T min), gte_restriction(T min)
  • min_excl_restriction(T min), gt_excl_restriction(T min)
  • max_restriction(T max), lte_restriction(T max)
  • max_excl_restriction(T max), lt_excl_restriction(T max)
  • discrete_restriction(std::initializer_list<T> values), discrete_restriction(std::array<T, SZ> values), discrete_restriction(std::vector<T> values)
