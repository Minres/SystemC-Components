Detailed Description

template<typename TYPES = chi::chi_protocol_types>
class chi::scv::chi_trx_recorder< TYPES >

The TLM2 transaction recorder.

This module records all TLM transaction to a SCV transaction stream for further viewing and analysis. The handle of the created transaction is storee in an tlm_extension so that another instance of the chi_recorder e.g. further down the path can link to it.
