Detailed Description

template<typename TYPES = axi::axi_protocol_types>
class axi::lwtr::axi_lwtr< TYPES >

The TLM2 transaction recorder.

This module records all TLM transaction to a LWTR transaction stream for further viewing and analysis. The handle of the created transaction is stored in an tlm_extension so that another instance of the axi_lwtr e.g. further down the path can link to it.
